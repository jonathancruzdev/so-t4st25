## Pasos de Git
    La primera vez configuro el usuario

**1.Inicializo un repositorio**
``` bash
    git init
```
**2.Configuramos el usuario**
``` bash
    git config user.name "Jonathan"
    git config user.email "jonathan@dv.edu.ar"
```

**3.Agregramos los archivos**
``` bash
    git add .
```
